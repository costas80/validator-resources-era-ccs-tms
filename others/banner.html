CCSTMS Validator

Validating RDF content against extended ERA ontology domains

Home API Docs Support

The CCSTMS Validator enables validation of RDF data against ERA's extended ontology, covering multiple domains such as infrastructure, rolling stock, engineering, and operations.

Validation is also available through:

  • REST API – validate directly via web requests
  • SOAP API – enterprise integration support
  • Command-line tool (CLI) – offline validation

Documentation and user guides are available here. For questions or feedback, please contact support.
